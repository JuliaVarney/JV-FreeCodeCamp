Introduction

So you’ve learned HTML and now you’re ready for your next steps? Welcome to an introduction to CSS! By learning CSS, you’ll spice up your dull HTML and make it into a beautiful page. Below you’ll learn the basics of changing colors, adjusting the font, and adjusting your text location. Let’s get started!

Before CSS

Before you learn CSS you should already know:

  • Where to code (i.e. CodePen or VSCode)
  • How to write HTML

First Things First

To start, you'll first need to know how to:

  • Use the style element
  • Target an element as a selector

Using the style element

While the HTML is in the body, the style will live in the head. You can add this below the title element and above the end of the head. You will simply add the style attribute and put all CSS within it.

Target an element as a selector

Your body may contain many elements such as a header, main, footer, div, etc. When defining what you would like to style, you will first want to identify your element and then place your changes between two brackets such as:

h1 { }

Adding Color

Color adds character to a page and you can adjust the color of many things. Here we will discuss:

  • Changing the color of words
  • Changing the background color
  • Using a color picker to so you can expand your horizons

Changing the color of words

First you’ll want to target which element you would like change the color of. As an example if we wanted to change the h1 from it’s standard black to red, we would first add an h1 selector. Then within the style element. Inside the brackets we will target the color by adding color: red;. Once you save and reload your page, you will see the change.

h1 { color: red; }

Changing the background color

Similar to changing the color of the word, you will first add the selector with the section of the page you would like to change. If you want the whole page a different color, you will target the body. As an example, we want to change the body from the standard white to blue. First we will add body { }. Within the the brackets, we will target the background-color: blue; like the example below:

body { background-color: blue; }

Using a Color Picker

While you can use standard colors such as red or blue, really the world is your oyster. By using a color picker such as Eye Dropper you can hover over colors on your screen and pick up the #, rgb, or hsl and plug it in as your color. For example:

p { color: #ffffff; color: hsl(0,0,100); color: rgb(255,255,255); }

Adjusting Font

When adjusting your font, you can manipulate:

  • Font Size
  • Font Weight
  • Font Type

Font Size

Depending on what you are targeting, some elements have predefined measures for a font such as h1 through h6. For this section we will focus on a paragraph. You can adjust the size by first using the selector p { }. Then use font-size: //px,em, etc.;

p { font-size: 10px; }

Font Weight

Similar to font size, within the brackets you will use font-weight: //bold, bolder, light, normal, or 100-900. Test it out and see what you like!

p { font-weight: 500; }

Font Type

When changing the font, you will use the property of font-family: ;. Here there are options such as Ariel available or you can go to Google Font and add fonts. To do this, you will choose the font you want in Google Font and select “Select this Family”. Here you will select “Embed”. Copy the link and paste it in your code under your title and before style. Then use the given font-family on the selector.

p { font-family: Georgia, 'Times New Roman', Times, serif; }

Changing Location

When you finish your HTML, all of your content is aligned left as a standard. You can change the location in CSS to help make your page flow better. While there are many ways to do this, the most basic way is to use the text-align feature. You will choose your select, for example your h1, which you may want in the center of your screen. Using text-align, you can choose left, right, center, etc. which would look like:

h1 { text-align: center; }

Try it Out!

That concludes a basic introduction to CSS. Take a moment to try out these features and bring your HTML to life!
